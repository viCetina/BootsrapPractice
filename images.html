Bootstarp Image Shape

Rounded Corners

The .img-rounded class adds rounded corners to an image (not avaible in IE8)

Circle

The .img-circle class shapes the images to a circle(IE8 does not support rounded corners)

Thumbnail

The .img-thumbnail class creates a thumbnail of the image

Image Responsive

The .img-responsive class makes the image scale nicely to the parent element

Image Gallery

The .thumbnail class can be used to display an image gallery.

The .caption class adds proper padding and a dark grey color to text inside thumbnails.

What is aspect ratio?

The aspect of an image describes the proportional relationship between its width and its height. Two common video aspect ratios are 4:3 (the universal video format of the 20th century), and 16:9 (universal for HD television an european digital television).
