Contextual Classes

Contextual clases cam be used to color table rows or table cells. The class that can be used are: .active, .success, .info, .warning, and .danger

The .active class applies the hover color to the table row or table cell.

The .success class indicates a succsesful or positive action.

The .info class indicates a neutral informative change or action.

The .warning class indicates a warning that maight need attention.

The .danger class indicates a dangerus or potentially negative action.
