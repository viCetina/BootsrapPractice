Closing Alert

To close the alert message, add a .alert-dismissable class to the container. The add class="close" and data-dismiss="alert" to a link or a button element (when you click on this the alert box will disappear).

The alert-dismissable class adds some extra padding to the close button.

The aria-* attribute and "×" explanation

To help improve accesibility for people using screen readers, you should include the aria-label="close" attribute, when creating a close button.

"×" (x) is an HTML entity that is the preferred icon for close buttons, rather than the letter "x"
